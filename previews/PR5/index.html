NlmeData

Documentation for NlmeData.

This package provides the data sets from the nlme package for R as Arrow files.

There are only two functions, datasets and dataset, defined, but not exported, in this package. The reason they are not exported is so that the names do not conflict with similar names in other packages.

NlmeData.dataset — Method
dataset(nm)

Return, as an Arrow.Table, the data set named nm, which can be a String or Symbol

source
NlmeData.datasets — Method
datasets()

Return a vector of names of the available data sets

source

The capitalization of the names of the datasets in the R package is preserved, even though it is more common to use lower case names in Julia.

An individual dataset always has a response variable and a grouping factor. For example, in the Rail dataset on the travel times of ultrasonic head-waves in 6 different samples of railway rails

dataset(:Rail)
Arrow.Table with 18 rows, 2 columns, and schema:
 :rail    String
 :travel  Int8

the response is travel and the grouping factor is rail.

Frequently there are other covariates recorded. For example, in a longitudinal data set one of the covariates will represent the time of the observation.

dataset(:Orange)
Arrow.Table with 35 rows, 3 columns, and schema:
 :tree           String
 :age            Int16
 :circumference  Int16

A convenient way to view an individual dataset is to create a GroupedDataFrame with DataFrames.groupby. (DataFrames is not a dependency of NlmeData - so as to avoid a heavyweight dependency. You must load it separately.)

using DataFrames
groupby(DataFrame(dataset(:Orange)), :tree)
